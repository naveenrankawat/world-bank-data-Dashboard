## 🌍 World Bank Pro Dashboard (v3)

An advanced interactive dashboard built using Python to analyze global economic and social indicators from World Bank datasets (1960–2018).

### 🚀 Key Features
- 📊 Interactive web-based dashboard using Dash  
- 📈 Time-series analysis of global indicators  
- 🏆 Top 15 countries comparison  
- 🔵 Scatter plot for multi-metric relationships  
- 📊 Distribution analysis (histogram + box plot)  
- 🥧 Region-wise contribution visualization  
- 🌎 Global choropleth map  

### 🎯 Smart Functionality

- Filter data by Year, Region, and Income Group  
- Compare multiple indicators dynamically  
- Auto-generated insights showing trends over time  
- Automatic CSV detection or manual upload option  
- Export filtered dataset as CSV
- 
### 🛠️ Tech Stack

- Python  
- Dash (Plotly)  
- Pandas & NumPy  
- Plotly Express & Graph Objects  
- Dash Bootstrap Components
